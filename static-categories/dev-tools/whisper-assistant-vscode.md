# whisper-assistant-vscode

开发工具 | cursor

Whisper Assistant is a powerful VSCode extension that allows developers to use their voice to develop software.  Speech can be transcribed locally using the Whisper model or remotely using an API such as OpenAI's.  By combining this capability with Cursor developers can quickly develop software using their voice to prompt AI models.
